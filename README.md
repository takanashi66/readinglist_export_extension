# Readlist Export Extension

Chromeのリーディングリストをサイドパネルで管理し、JSON形式でエクスポートできるChrome拡張機能です。

## 概要

この拡張機能は、ブラウザ標準のサイドパネルを利用してリーディングリストを表示します。リストの閲覧、検索、追加、削除、そしてJSON形式でのエクスポート機能を提供し、リーディングリストの管理を効率化します。

## 主な機能

- **サイドパネル表示**: ブラウザのサイドパネルでリーディングリストを常時表示可能です。
- **リストの閲覧**: 未読・既読アイテムを一覧表示します。Favicon、タイトル、ドメイン、登録日時が表示されます。
- **アイテムの検索**: タイトルやURLでリスト内をフィルタリング検索できます。
- **現在のタブを追加**: ワンクリックで閲覧中のページをリーディングリストに追加できます。
- **JSONエクスポート**: リーディングリスト全体をJSONファイルとしてダウンロードできます。
- **既読・未読管理**:
  - リストアイテムをクリックすると、ページを開くと同時に「既読」状態（文字色グレー・細字）になります。
  - 既読アイテムには「未読に戻す」ボタン（チェックマークアイコン）が表示され、クリックすると未読状態に戻ります。
- **削除**: 各アイテムの右側にあるゴミ箱アイコンをクリックして削除します。
- **UI**: 削除・未読ボタンは円形で、ホバー時に背景色が変化します。既読アイテムは視覚的に区別しやすいスタイルが適用されます。

## インストール方法

1. このリポジトリをクローンまたはダウンロードします。
2. Chromeブラウザで `chrome://extensions` を開きます。
3. 右上の「デベロッパーモード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、このディレクトリを選択します。

## 使用方法

1. 拡張機能のアイコンをクリックするか、サイドパネルを開いて「Readlist Export Extension」を選択します。
2. **リストの表示**: 保存されたリーディングリストが表示されます。
3. **追加**: 右上の「＋」アイコンをクリックすると、現在アクティブなタブがリストに追加されます。
4. **エクスポート**: 右上のダウンロードアイコンをクリックすると、`reading_list.json` がダウンロードされます。
5. **検索**: 上部の検索バーにキーワードを入力してアイテムを絞り込みます。
6. **削除**: 各アイテムの右側にあるゴミ箱アイコンをクリックして削除します。

## 技術仕様

- **Manifest V3**: 最新のChrome拡張機能仕様に準拠しています。
- **Permissions**:
  - `sidePanel`: サイドパネルUIの使用
  - `readingList`: リーディングリストの読み取り・変更
  - `tabs`: 現在のタブ情報の取得（追加機能用）

## ファイル構成

- `manifest.json`: 拡張機能の設定ファイル
- `background.js`: バックグラウンドプロセス（サイドパネルの制御）
- `panel.html`: サイドパネルのHTML構造
- `panel.css`: サイドパネルのスタイル定義
- `panel.js`: UIロジックとインタラクションの実装
