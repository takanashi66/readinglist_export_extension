# Readlist Export Extension 仕様書

Google ChromeのリーディングリストをエクスポートするChrome拡張機能です。

## 概要

この拡張機能は、ユーザーのChromeリーディングリストを取得し、JSON形式でダウンロードする機能を提供します。

## 機能

1.  **ポップアップ表示**: 拡張機能のアイコンをクリックすると、ダウンロードボタンを含むポップアップが表示されます。
2.  **ダウンロード機能**: 「Download」ボタンをクリックすると、リーディングリストの内容が記載されたJSONファイルがダウンロードされます。

## 出力フォーマット

ダウンロードされるファイルは以下のJSON形式です。

```json
[
  {
    "title": "ページタイトル1",
    "url": "https://example.com/page1"
  },
  {
    "title": "ページタイトル2",
    "url": "https://example.com/page2"
  }
]
```

- **title**: ページのタイトル
- **url**: ページのURL
